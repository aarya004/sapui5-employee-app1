# SAPUI5 Employee App

This is a simple SAPUI5 application built using MVC architecture.

## Features
- SAPUI5 MVC Structure
- JSON Model Data Binding
- Employee Information Display
- Responsive UI using sap.m library

## Technologies Used
- SAPUI5
- JavaScript
- HTML
- JSON
- MVC Architecture
